# Sequence Viewer

Canvas view of the template sequence with annotations.

## Markers

- **CDS boundaries**: vertical bars at start and end of the selected gene
- **Mutation positions**: coloured markers on residues being mutated
- **Primer windows**: highlighted segments when a Result Table row is selected

## Zoom & scroll

- Mouse wheel to zoom around the cursor
- Click-and-drag to pan
- Double-click a marker to centre

## Region labels

Domains fetched from InterPro (see [Diversity Strategies](diversity-strategies.md)) render as translucent bands above the sequence.
