# Custom Polymerase Editor

![Custom Polymerase Editor dialog](../screenshots/14-polymerase-editor.png)

Create or modify polymerase profiles persisted to `~/.kuro/custom_polymerases.json`.

## Open

Parameter panel → **Custom Polymerase** button.

## Fields

- **Name** (unique, becomes dropdown label)
- **Manufacturer**
- **Tm formula**: SantaLucia98 / OligoCalc / Primer3
- **Default Tm targets**: fwd / rev / overlap
- **GC range**: min / max
- **Primer length**: fwd-min/max, rev-min/max
- **Salt correction**: Na⁺ mM, Mg²⁺ mM, dNTP mM
- **DMSO %** (optional)

## Clone / Delete

Clone from any bundled profile as a starting point. Delete removes the custom entry; bundled profiles cannot be deleted.
