# 키보드 단축키

## 전역

| 단축키 | 동작 |
|---|---|
| `Ctrl/Cmd+O` | 서열 파일 열기 |
| `Ctrl/Cmd+S` | 워크스페이스 저장 |
| `Ctrl/Cmd+Shift+S` | 다른 이름으로 저장 |
| `Ctrl/Cmd+E` | Excel 내보내기 |
| `Ctrl/Cmd+R` | Design Report 열기 |
| `Enter` | Design Primers (텍스트 입력 외부 포커스 시) |
| `Esc` | 현재 설계 취소 / 다이얼로그 닫기 |

## 결과 테이블

| 단축키 | 동작 |
|---|---|
| 컬럼 헤더 클릭 | 오름차순 / 내림차순 정렬 |
| Fwd / Rev 셀 클릭 | 후보 popover 열기 |
| HP 배지 클릭 | hairpin / dimer 세부 열기 |

## 드래그 앤 드롭

`.gb` / `.dna` / `.fa` 파일을 창에 드롭하면 로드.

*스텁 — 단축키 레퍼런스 카드 추가 예정.*
