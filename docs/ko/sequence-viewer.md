# 서열 뷰어

템플릿 서열에 주석을 덧입힌 canvas 뷰.

## 마커

- **CDS 경계**: 선택된 유전자의 시작·종료 위치에 세로선
- **변이 위치**: 변이 대상 잔기에 색상 마커
- **프라이머 윈도우**: Result Table 행 선택 시 해당 프라이머 구간 하이라이트

## 줌·스크롤

- 휠로 커서 기준 줌
- 드래그로 팬
- 마커 더블클릭 시 중앙 정렬

## 영역 라벨

InterPro에서 가져온 도메인([다양성 전략](diversity-strategies.md))은 서열 위 반투명 밴드로 렌더.

*스텁 — 뷰어 스크린샷 추가 예정.*
