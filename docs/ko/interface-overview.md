# 인터페이스 개요

Kuma는 하나의 프로젝트 작업 공간에서 연결된 두 도구를 제공한다.

- **Kuro**는 점수화된 단백질 변이를 프라이머 설계, 플레이트 배치 및 구조 기반 검토 결과로 변환한다.
- **Mame**은 시퀀싱 판정과 활성 측정값을 결합하여 다음 학습 라운드에 사용할 정리된 테이블을 만든다.

## 공통 작업 공간

최상단 앱 표시줄에서 **Kuro**와 **Mame**을 전환한다. 바로 아래 프로젝트 표시줄에는 활성 프로젝트, 프로젝트 단계 및 자동 저장 상태가 표시된다. 도구를 전환해도 결과는 현재 프로젝트에 연결된 상태를 유지한다.

두 도구는 동일한 3영역 레이아웃을 사용한다.

1. **워크플로 레일(왼쪽)** — 순서가 정해진 단계와 현재 진행 위치.
2. **주 작업 영역(중앙)** — 입력, 결과표, 플레이트, 서열·구조 뷰 및 현재 단계의 주요 작업.
3. **인스펙터(오른쪽)** — 현재 단계의 매개변수, 선택 항목 상세 정보, 경고 및 내보내기 설정.

하단 상태 표시줄에는 sidecar 상태와 장시간 작업의 진행 상황이 표시된다.

## 가이드 투어

새로 만든 프로젝트에서는 짧은 spotlight 투어가 시작된다.

1. 프로젝트 개요와 Kuro/Mame 전환 방법.
2. Kuro 워크플로 레일, 주 작업 영역 및 인스펙터.
3. 해당 프로젝트에서 Mame에 처음 진입할 때 별도의 Mame 투어.

설명이 열려 있는 동안 강조된 영역은 일시적으로 조작할 수 없다.

- **이전 / 다음 / 완료**로 현재 투어를 이동한다.
- **모든 투어 건너뛰기**는 해당 프로젝트의 모든 자동 투어를 비활성화한다.
- `Esc`는 현재 투어만 닫으며 영구적으로 비활성화하지 않는다.
- **도움말 → 가이드 투어 보기**에서 현재 탭의 투어를 다시 실행할 수 있다.
- 기존 프로젝트에는 투어가 자동 표시되지 않는다.

Kuro 설계와 wet-lab 실행 후 Mame으로 전환하여 클론을 검증하고, 활성 데이터를 병합하고, 다음 라운드 학습 입력을 준비한다.
